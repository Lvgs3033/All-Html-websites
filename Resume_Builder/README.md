# 🤖 AI-Powered Resume Builder

A modern, interactive resume builder with stunning popup animations, AI-powered suggestions, and professional PDF generation. Built with vanilla HTML, CSS, and JavaScript.

![Resume Builder Preview](https://img.shields.io/badge/Status-Active-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

### 🎭 **Interactive Popup System**
- **10 Different Popup Types** with unique animations
- **6 Animation Styles** (Bounce, Slide, Flip, Zoom, Fade-Scale, Elastic)
- **Smart Positioning** and responsive design
- **Keyboard Support** (ESC to close, Enter to confirm)
- **Click Outside** to dismiss functionality

### 🤖 **AI-Powered Suggestions**
- Smart recommendations for each resume section
- Industry-specific skill suggestions
- Professional summary templates
- Job position and experience suggestions

### 🎨 **Modern Design**
- **Pink Gradient Theme** with dark mode support
- **Smooth Animations** and micro-interactions
- **Responsive Design** for all devices
- **Real-time Preview** with live updates
- **Progress Tracking** visual indicator

### 📄 **Professional Output**
- **High-Quality PDF Generation** with optimized fonts
- **Print-Friendly** styling
- **Multiple Export Options**
- **Font Optimization** for better readability

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required!

### Installation

1. **Clone or Download** the project:
\`\`\`bash
git clone https://github.com/dhvani1215/ai-resume-builder.git
cd ai-resume-builder
\`\`\`

2. **Open in Browser**:
\`\`\`bash
# Simply open index.html in your browser
open index.html
# or
double-click index.html
\`\`\`

3. **Start Building**:
   - Fill in your personal information
   - Add skills, education, and experience
   - Use AI suggestions for better content
   - Download your professional resume as PDF

### Technologies
   - Html5,css and vanila js
